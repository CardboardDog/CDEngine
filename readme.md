# CDEngine
![image](https://github.com/CardboardDog/CDEngine/blob/main/img/CubeDemo.gif?raw=true)
</br>
A 3D Engine using IRRLICHT, where the goal is realistic graphics and decent performance.
## Features
- objects
- scenes
- skyboxes
- debugger
- cameras
- transformations
## Planned Features

- Ui

- Realtime Lighting
- Shadows and shaders
- Particles
- Scripting Language
